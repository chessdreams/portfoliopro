1 · First import

~5 min

  1. Export a full JSON backup from desktop PortfolioPro Settings.
  2. Open Settings here → Import → choose the file.
  3. Open the Dashboard and confirm CMB rows appear.
  4. Export regularly to keep an off-browser backup.

2 · Deposits & returns

~3 min

  1. Open Transactions.
  2. Add deposits with description CT, SL, or FC.
  3. Return to the dashboard — check Return % (dep.) columns.

3 · Compare snapshots

~4 min

  1. Select two CMB dates on the dashboard.
  2. Click Compare in the selection bar.
  3. Review holdings diff, added/removed/modified counts.
  4. Or use More → Compare → Latest CMB pair.

Step-by-step guides

Price trends

Select 2+ CMB dates under More → Price trends. Compare avg and last prices per symbol.

Trading journal

Pick CMB snapshots for FIFO realized P&L, win rate, and export/print.

Target prices

Choose a portfolio, pick an institute chip, edit targets per holding.

Stock calculator

Average-down and scenario calculations; saves linked to latest CMB.

Trade log upload

One-off Excel trade summary — drag & drop without syncing portfolios.

Backup & restore

Full JSON export from Settings before browser changes or imports.

Tips

  • Use card vs table toggle and column sorting to scan snapshots quickly.
  • Hard-refresh (Cmd+Shift+R) after app updates if styles or scripts look stale.
  • Journal and trends require CMB — filter combined-only on the dashboard first.
  • Export JSON regularly from Settings — online mode has no Excel auto-sync.
