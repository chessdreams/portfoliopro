Getting started

PortfolioPro Online tracks Colombo Stock Exchange portfolios from JSON backups exported by the desktop app. Data stays in your browser (IndexedDB) — nothing is sent to a cloud server.

Quick setup

  1. In desktop PortfolioPro, open Settings and export a full JSON backup.
  2. In this online app, open Settings → Import and choose that file.
  3. Open the Dashboard — CMB (combined) snapshots should appear.
  4. Add deposits in Transactions (use CT, SL, or FC tags) for return-on-deposits metrics.
Default dashboard view shows CMB (combined) only — one row per trade date with all accounts rolled up.

Import & backup

PortfolioPro Online does not read Excel folders from disk. Use JSON export/import to move data between browsers or from the desktop app.

  • Export a full backup from Settings before clearing data or switching browsers.
  • Import JSON restores portfolios, transactions, targets, and calculator saves.
  • Dashboard Import accepts single JSON files or ZIP bundles.
  • For broker Excel trade summaries, use Trade log upload (browser-only).

For automatic sync from Portfolio*.xlsx files, use the desktop app in portfoliopro/, then export and import here.

Dashboard

The dashboard lists portfolio snapshots with card or table view, sorting, and multi-select.

Selection actions

  • Compare — exactly 2 snapshots (same account key, e.g. CMB↔CMB).
  • Trends — 2+ CMB dates for price charts.
  • Journal — opens the FIFO trading journal for selected CMB dates.

Open a single snapshot with View for holdings, sector charts, and scoped activity.

More menu tools

Find these under the header More ⋮ menu:

  • Compare — side-by-side holdings diff between two dates.
  • Price trends — average and last price across CMB dates (pick 2+ portfolios).
  • Journal — FIFO buy/sell reconstruction from synced data; sell P&L includes 1.12% fees.
  • Trade log — standalone Excel upload for broker trade-summary files (no sync required).

Targets

Target management sets exit prices per holding in a chosen portfolio, with forecast P&L (1.12% sell fee on proceeds).

Symbol targets is a quick ticker-level list shared with portfolio targets. Use institute chips on the management page for per-broker target sets.

Transactions

Record net deposits and withdrawals. Description tags map to accounts:

  • CT → DSA
  • SL → ARR
  • FC → HDF

Cumulative deposits drive Return % (dep.) on the dashboard and portfolio view.

Settings & backup

  • Export full JSON backup or individual data sets (portfolios, transactions, targets, calculators).
  • Import JSON to restore; destructive clear actions are available with confirmation.
Export a full backup before importing or clearing data.

Troubleshooting

Dashboard empty after import

Confirm the JSON file contains portfolio records. Try a full backup export from desktop PortfolioPro Settings, not a partial export.

Compare or trends disabled

Compare needs exactly 2 portfolios with the same key. Trends and journal require CMB snapshots — select 2+ combined dates.

Data missing after browser change

Data is per-browser. Export from Settings on the old browser and import on the new one.
