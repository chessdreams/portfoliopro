Portfolio Comparison Mode:

How to use comparison mode:

  1. Set the portfolio date using the date picker
  2. First upload: Your initial portfolio (e.g., yesterday's data)
  3. Second upload: Your new portfolio (e.g., today's data)
  4. Dashboard will show comparison charts and change indicators
  5. Portfolios are automatically sorted by date for proper comparison order

Multiple File Upload with Date Verification:

  • Select multiple portfolio files using Ctrl+Click (or Cmd+Click on Mac)
  • Click "Preview & Verify Dates" to see extracted dates for each file
  • Use the "Sort by" dropdown to organize files for easy review
  • Verify and modify dates as needed using the date picker
  • Portfolio Key: Enter a key (e.g., HDF, DSA, ARR) for each file
  • Auto Key Extraction: Keys are automatically extracted from A1 cell content (text between "(" and "/" characters)
  • Duplicate Detection: Files with same dates are highlighted in red
  • Individual Rejection: Click "❌ Reject" to exclude specific files
  • Bulk Rejection: Click "🗑️ Reject All Duplicates" to keep only first file per date
  • Click "Confirm & Upload" to process accepted files with verified dates
  • Dual View: See both individual portfolios by key and combined portfolios by date
  • Dashboard shows portfolios organized by date with individual keys and combined totals

Expected File Format:

  • Security - Stock symbol/ticker (e.g., ALUM.N0000)
  • Quantity - Number of shares
  • Avg Price - Average purchase price
  • Total Cost - Total cost basis
  • Market Value - Current market value
  • Unrealized Gain / (Loss) - Unrealized profit/loss
  • Unrealized Gain/Loss % - Return percentage

Trade Execution Summary (Optional):

  • Upload your broker's trade execution summary Excel file
  • Provides accurate buy/sell prices and transaction details
  • Enables precise P&L calculations with exact sold prices
  • Creates detailed trading journal with individual transactions
  • Expected columns: Security, Side (BUY/SELL), Avg Price, Executed Quantity, Gross Amount

Supported Formats:

  • CSV files - Comma-separated values
  • Excel files - .xlsx and .xls formats
  • Multiple sheets - Excel files with multiple worksheets are supported
