Import a JSON backup from desktop PortfolioPro in Settings → Import, or use dashboard Import for JSON/ZIP files. The trade log page accepts a separate Excel trade-summary upload in the browser.
Each Excel file maps to an account (DSA, ARR, HDF). Snapshots on the same date are also rolled into CMB (combined). Compare, trends, and journal use CMB by default. Switch the table filter to “All” to see individual accounts.
Select exactly two snapshots on the dashboard (checkboxes), then click Compare. Or open More → Compare and pick baseline vs compare dates. Keys must match (CMB↔CMB, etc.).
Journal (More menu) builds FIFO trades from imported portfolio snapshots — best for ongoing CMB analysis. Trade log is a one-off Excel upload for broker trade-summary files.
